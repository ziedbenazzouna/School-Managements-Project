# School Management Solution

Development of a web-based school management solution:

* User Registration.
* JWT User Authentication.
* Configuration JWT.
* Create Login Web Method.
* Apply Authorization to all Web Methods.
* Authorize with Role, Claim & Policy.
* Design an Angular Forms for SignIn, SignUp and school management.
* Angular Logout.
* CRUD Operation.
* Make Http Request from Angular to a Protected Web API Endpoint.
* Protect Angular Routes based user claims and roles
* Angular Form Validation.
* Angular Animation.
* Toast Notification.


**Technical environments:** 
.NET 8, REST Web API, C#, Angular 18, Visual Studio 2022, Visual Studio Code, SQL Server.
